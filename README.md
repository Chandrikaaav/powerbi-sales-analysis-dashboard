# Sales Analysis Dashboard (Power BI)

## Project Overview
This project presents a Sales Analysis Dashboard built using Power BI to analyze sales performance across products, regions, and salespersons.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales Trend Over Time
- Sales by Region
- Top Selling Products
- Category Contribution
- Sales vs Profit Analysis
- Interactive Slicers for Date, Region, Category, Customer Segment

## Tools Used
- Microsoft Power BI
- CSV Dataset
- Data Visualization

- sales-analysis-dashboard.pbix
